# ReachNote 品牌素材包

本素材包基于“年轻、清爽、工具感、低 AI 味”的方向生成，适合 macOS 菜单栏应用、桌面应用图标、官网 favicon、产品文档与演示材料。

## 文件结构

- `svg/`：可缩放矢量源文件
- `png/icon/`：应用图标与透明符号，多尺寸导出
- `png/logo/`：横版 / 竖版 Logo PNG
- `favicon/`：16/32/48 PNG 与 `.ico`
- `docs/`：品牌说明与导出清单

## 推荐使用

- App 图标：`png/icon/reachnote-icon-1024.png`
- macOS 菜单栏 / 小尺寸：`png/icon/reachnote-icon-32.png` 或 `reachnote-symbol-transparent-32.png`
- 官网 Header：`svg/reachnote-logo-horizontal.svg`
- 深色背景：`svg/reachnote-logo-horizontal-dark.svg`
- Favicon：`favicon/reachnote-favicon.ico`

## 色彩

- 主色：`#6366F1`
- 辅助紫：`#A78BFA`
- 深色文字：`#0F172A`
- 浅色背景：`#F5F6FA`

## 注意

SVG 中的文字 Logo 使用系统无衬线字体 fallback，不包含任何字体文件。若要完全固定字形，建议在 Figma / Illustrator 中将文字转曲后再用于生产。
